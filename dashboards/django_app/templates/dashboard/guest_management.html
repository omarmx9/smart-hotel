Create Guest User Manage Groups

How to Create a Guest Account

1. Create a new user in Authentik
2. Add them to the smart-hotel-guests group
3. Set custom attributes: room_number and expires_at
4. Share credentials with the guest (Authentik can send via email/SMS)
